# Snake and Ladders

A classic Snake and Ladders game implementation with modern features and extensible design patterns.

## Overview

Snake and Ladders is a timeless board game where players race to reach the end of a board (usually 100 squares) by rolling dice. Landing on a ladder takes you forward, while landing on a snake sends you back. The first player to reach or exceed the final square wins!

## Features

- **Classic Gameplay**: Traditional snake and ladders mechanics
- **Multiplayer Support**: Play with 2-6 players
- **Dice Rolling**: Realistic dice simulation
- **Board State Management**: Real-time board tracking and updates
- **Win Detection**: Automatic winner detection
- **Replay Option**: Play multiple rounds easily

## Game Rules

1. Players take turns rolling a die (1-6)
2. Move forward by the number rolled
3. Landing on a ladder climb moves you up
4. Landing on a snake head sends you down to the tail
5. First player to reach or exceed square 100 wins

## Architecture

This project implements Low-Level Design (LLD) patterns including:

- **Board Management**: Handling board state and square positions
- **Player Management**: Tracking player positions and turns
- **Game Engine**: Core game logic and flow control
